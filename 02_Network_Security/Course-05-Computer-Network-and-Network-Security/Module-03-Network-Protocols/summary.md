In this module, I explored how different network protocols actually control communication between systems. I started with the transport layer, where I learned the difference between TCP and UDP. Understanding how TCP ensures reliable communication while UDP is faster but less reliable helped me clearly see where each is used.

Using Wireshark in the lab made things much more practical, as I could actually observe TCP and UDP traffic instead of just reading about it. I also got introduced to endpoint protection using OpenEDR, which showed how monitoring happens at the system level.

Moving to the application layer, I learned how protocols like DNS and DHCP work behind the scenes. DNS filtering and DHCP configuration labs helped me connect theory with real implementation. Concepts like syslog, port mirroring, and network flow analysis gave me a clearer idea of how monitoring and analysis are done in real networks.

Overall, this module helped me understand how protocols operate across layers and how they are used in both communication and security monitoring.
