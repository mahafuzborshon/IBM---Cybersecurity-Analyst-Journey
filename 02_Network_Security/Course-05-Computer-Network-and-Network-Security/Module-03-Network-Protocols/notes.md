* I understood that protocols define how communication happens between systems

* Transport Layer:

  * TCP = reliable, connection-based (used where accuracy matters)
  * UDP = faster, no guarantee (used for speed-critical tasks)
  * I observed TCP & UDP packets using Wireshark → helped me see real traffic behavior

* I learned how endpoints can be monitored and protected using tools like OpenEDR

* Application Layer:

  * DNS translates domain names into IP addresses
  * DHCP automatically assigns IP addresses to devices
  * I practiced DHCP setup → understood how devices get IPs dynamically

* DNS Filtering:

  * Helps block malicious or unwanted domains
  * Useful for basic network security control

* Syslog:

  * Used for logging events across network devices
  * Important for monitoring and incident tracking

* Network Analysis:

  * Flow analysis helps understand traffic patterns
  * Port mirroring allows capturing traffic for inspection
  * Promiscuous mode lets a device capture all packets in a network

* User Behavior Analysis:

  * Helps detect unusual or suspicious activities

* This module showed me how protocols are not just for communication, but also critical for monitoring, analysis, and security operations
