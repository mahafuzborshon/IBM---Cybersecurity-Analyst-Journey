In this module, I started building my understanding of how computer networks actually work at a fundamental level. I explored how devices communicate with each other, how data travels across networks, and the role of different network components in enabling communication.

I learned about key concepts like IP addressing, MAC addresses, and how data is broken down into packets before being transmitted. Understanding the difference between LAN, WAN, and other network types helped me visualize how networks scale from small local setups to global systems like the internet.

I also got introduced to basic networking models like TCP/IP and how protocols define the rules for communication. This module gave me a solid base to understand how networks operate behind the scenes, which is essential before diving deeper into network security concepts.
