In this module, I deepened my understanding of how networks actually identify devices and move data efficiently. I explored IP addressing in detail, including how binary plays a role behind the scenes and how IP addresses are structured across different network classes.

I learned how routing works, how routers make decisions using routing tables, and how data finds the best path from source to destination. The concept of Layer 2 and Layer 3 addressing helped me clearly understand the difference between MAC and IP-level communication.

I also got introduced to important protocols like ARP, which connects IP addresses to physical MAC addresses. Learning about IPv6 gave me insight into how modern networks are evolving beyond IPv4 limitations. Overall, this module helped me connect the dots between addressing, routing, and switching, making network communication much clearer from a practical perspective.
