* I understood firewall basics and how they control incoming/outgoing traffic

* Stateless vs Stateful:

  * Stateless = checks packets individually
  * Stateful = tracks active connections → more secure

* Firewalls can be combined with intrusion systems for better protection

* IDS vs IPS:

  * IDS = detects and alerts
  * IPS = detects + actively blocks threats

* I learned NAT (Network Address Translation):

  * Hides internal IP addresses
  * Adds an extra layer of security

* Windows Firewall labs:

  * I practiced creating and managing firewall rules
  * Helped me understand real traffic control

* Limitations:

  * Firewalls and IDS cannot stop everything → need layered security

* NAC (Network Access Control):

  * Controls which devices can access the network

* DLP (Data Loss Prevention):

  * Prevents sensitive data from leaving the network

* EDR (Endpoint Detection & Response):

  * Monitors endpoint activities and detects threats

* XDR (Extended Detection & Response):

  * Combines multiple security layers for better detection

* File Integrity Monitoring:

  * Detects unauthorized file changes

* This module showed me that security is not one tool → it’s a combination of multiple systems working together
