In this project, I applied everything I learned from networking and security modules into a practical scenario. I designed a network for TechSafe Ltd by dividing it into three departments: Administration, Sales, and IT, each with its own subnet. This helped me clearly understand how subnetting works in real environments.

While calculating subnet addresses and subnet masks, I realized how important proper IP planning is for network organization and scalability. It was a bit challenging at first, but mapping the ranges made the concept much clearer.

Using Wireshark was one of the most interesting parts. I captured HTTPS traffic while browsing sites like Google and Wikipedia and applied filters to isolate relevant packets. This gave me real visibility into how encrypted web traffic flows.

Finally, configuring firewall rules in Windows Defender helped me understand how organizations restrict unwanted traffic like HTTP and FTP. Overall, this project connected theory with real-world implementation and strengthened my practical networking and security skills.
