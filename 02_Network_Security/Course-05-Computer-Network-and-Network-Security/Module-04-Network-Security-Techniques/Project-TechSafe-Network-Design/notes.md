* Task 1: Network Design

  * I created a network structure with 3 departments: Administration, Sales, IT
  * Each department was separated using its own subnet
  * Helped me visualize segmentation in real networks

* Task 2: Subnet Calculation

  * Base network: 198.51.100.0/24
  * Divided into 3 subnet ranges:

    * Administration: 198.51.100.1 – 62
    * Sales: 198.51.100.65 – 126
    * IT: 198.51.100.129 – 190
  * I calculated subnet addresses and masks
  * Understood how subnetting organizes IP allocation

* Task 3: Wireshark Analysis

  * Captured live traffic while browsing Google & Wikipedia
  * Applied filter: tcp.port == 443
  * Used additional filters to isolate specific domains
  * Observed HTTPS encrypted traffic behavior
  * Learned how analysts monitor network activity

* Task 4: Firewall Configuration

  * Used Microsoft Windows Defender Firewall
  * Blocked HTTP (port 80) and FTP (port 21) traffic
  * Understood how firewall rules control traffic

* Key Takeaway:

  * This project helped me combine subnetting, traffic analysis, and firewall configuration in one workflow
  * Gave me a practical understanding of real-world network security implementation
