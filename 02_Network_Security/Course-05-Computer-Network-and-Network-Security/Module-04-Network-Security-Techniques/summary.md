In this module, I focused on how networks are actually protected in real-world environments. I started by understanding how firewalls work, especially the difference between stateless and stateful inspection. It became clear how stateful firewalls track ongoing connections, making them more effective for security.

I explored IDS and IPS systems and how they detect and prevent malicious activities. The comparison between IDS and IPS helped me understand when monitoring is enough and when active blocking is needed. Learning about NAT also showed me how internal networks stay hidden from external exposure.

The hands-on labs with Windows Firewall made things much more practical, especially configuring rules and understanding how traffic is controlled.

In the second part, I learned about advanced security techniques like NAC, DLP, EDR, and XDR. These concepts showed me how modern security goes beyond just networks and includes endpoints and user activity. Overall, this module gave me a strong understanding of layered security and how multiple systems work together to protect a network.
