In this module, I worked with SQL practically and started understanding how databases are actually managed. I learned how to retrieve data using SELECT and explored filters like COUNT, DISTINCT, and LIMIT, which made querying more efficient.

I also practiced INSERT, UPDATE, and DELETE operations, which helped me understand how data is modified inside a database. The labs gave me hands-on experience, making everything much clearer.

Beyond SQL, I learned about database roles, permissions, and backup processes. This part made me realize how important access control and data recovery are in real-world systems. Overall, this module helped me connect database usage with management and security aspects.
