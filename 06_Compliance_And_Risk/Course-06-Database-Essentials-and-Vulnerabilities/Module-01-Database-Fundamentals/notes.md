- Databases are used to store, manage, and retrieve structured data efficiently

- Relational Databases
  - Use tables (rows & columns)
  - Structured schema
  - Relationships between tables using keys
  - Example concepts: primary key, foreign key

- Non-Relational Databases (NoSQL)
  - Flexible schema
  - Store data as JSON, key-value, documents
  - Better for unstructured or large-scale data

- Relational Data Concepts
  - Table → collection of data
  - Row → single record
  - Column → attribute/field
  - Keys → used to identify and link data
