In this module, I learned about one of the most critical security risks in databases— injection vulnerabilities. I started with the basics of injection flaws and then explored OS command injection in detail, which showed how attackers can execute system-level commands if input is not handled properly.

I then moved into SQL injection, which was the most important part. I understood how attackers manipulate queries to access or modify database data. The hands-on lab was very helpful because I got to see how these attacks actually work in practice.

I also explored different types of injection attacks and how they vary depending on the system. Overall, this module helped me understand how dangerous improper input handling can be and why secure coding practices are essential.
