- Injection Vulnerabilities
  - Occur when user input is not properly validated
  - Allows attackers to manipulate system behavior

- OS Command Injection
  - Executes system-level commands through input fields
  - Can lead to full system compromise
  - Covered in multiple parts for deeper understanding

- SQL Injection
  - Manipulates database queries
  - Can bypass authentication
  - Can read, modify, or delete database data

- Database Injection Vulnerabilities
  - Weak input validation leads to exploitation
  - Affects database integrity and confidentiality

- Types of Injection Attacks
  - SQL injection
  - OS command injection
  - Other injection types depending on system
