In this module, I learned how to apply security best practices to protect systems and data in real-world scenarios. I understood how strong password management and authentication methods like SSO help secure user access.

I explored how access control, authorization, and authentication work together to restrict unauthorized users. I also learned about device hardening and how it reduces vulnerabilities in systems.

Through hands-on labs, I practiced enforcing strong password policies and configuring Windows Firewall, which gave me practical exposure to system-level security.

I also gained knowledge about encryption concepts, email security, and how to identify phishing attempts. Overall, this module helped me understand how to actively protect systems using practical security measures.
