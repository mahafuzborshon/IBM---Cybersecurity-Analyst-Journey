- Identity and Access Management (IAM):
  - Manages user identities and permissions
  - Ensures only authorized users access systems

- Authentication:
  - Verifies user identity
  - Examples: passwords, biometrics

- Authorization:
  - Determines what a user can access
  - Happens after authentication

- Access controls:
  - Define rules for accessing resources
  - Based on roles and permissions

- Multi-Factor Authentication (MFA):
  - Uses multiple verification methods
  - Increases security

- Two-Factor Authentication (2FA):
  - A type of MFA with two verification steps

- Single Sign-On (SSO):
  - One login for multiple systems
  - Improves user convenience

- Passkeys (FIDO):
  - Passwordless authentication method
  - More secure and phishing-resistant

- Physical threats:
  - Theft of devices
  - Unauthorized physical access
  - Environmental risks

- Physical security controls:
  - Locks and surveillance
  - Access badges
  - Secure facilities

- Importance of combining physical and digital security

- Security is stronger when multiple layers are applied
