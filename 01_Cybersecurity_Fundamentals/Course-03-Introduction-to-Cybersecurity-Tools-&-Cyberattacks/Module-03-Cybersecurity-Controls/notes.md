- Internet attacks target networks and systems

- Mapping:
  - Attackers gather information about systems and networks
  - Helps identify weak points

- Packet sniffing:
  - Captures data packets from network traffic
  - Can expose sensitive information

- IP spoofing:
  - Attacker disguises identity using fake IP address
  - Used to bypass security or launch attacks

- Denial-of-Service (DoS):
  - Overloads system with traffic
  - Makes services unavailable

- Injection attacks:
  - Malicious code inserted into applications
  - Targets databases and input fields

- Cybersecurity controls:
  - Measures to protect systems and data

- Types of controls:
  - Preventive controls
  - Detective controls
  - Corrective controls

- System security:
  - Protects individual devices and operating systems

- Network security:
  - Protects data during transmission
  - Uses firewalls and monitoring tools

- Application security:
  - Built into software design
  - Focus on secure coding practices

- Vulnerability management:
  - Identifying and fixing security weaknesses
  - Regular scanning and updates

- Incident response:
  - Detecting and handling security incidents
  - Includes recovery and analysis

- Digital forensics:
  - Investigating cyber incidents
  - Collecting and analyzing evidence

- Security requires continuous monitoring and improvement
